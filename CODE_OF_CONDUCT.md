# Code of Conduct

## Our commitment

SmartTime AI is committed to a respectful, inclusive, and constructive open-source community.

We want this project to be welcoming to educators, developers, contributors, and users from different backgrounds and experience levels.

## Expected behavior

Please:
- be respectful and patient
- assume good intent, while staying honest and clear
- give actionable feedback
- focus on what helps the project and community move forward
- be especially considerate when discussing mistakes, bugs, or disagreements

## Unacceptable behavior

This includes:
- harassment or personal attacks
- hateful or discriminatory language
- trolling, intimidation, or deliberate disruption
- publishing private information without permission
- repeated bad-faith behavior in issues, pull requests, or discussions

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, or contributions that do not align with this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, please contact the maintainer through GitHub.

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, discussions, and review conversations.
