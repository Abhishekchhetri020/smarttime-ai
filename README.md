# SmartTime AI

Phase 1 scaffold for production timetable platform.

## Structure
- `backend/functions` Firebase Functions API layer
- `solver` FastAPI-based constraint solver service
- `admin-web` Next.js admin panel (scaffold)
- `android-app` Flutter app scaffold docs
- `infra` deployment/env templates
- `docs` project docs

## Phase 1 goals completed
- Repository structure and module boundaries
- Backend API skeleton with role middleware and health endpoints
- Solver service skeleton with job input/output contracts
- Admin panel bootstrap scaffold
- Integration contract docs
