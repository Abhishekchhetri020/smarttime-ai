# Support

## Getting help

If you need help with SmartTime AI:

- open a GitHub issue for bugs, feature requests, or documentation gaps
- include enough detail for someone else to reproduce the problem
- use screenshots, logs, and environment details when relevant

## Before opening an issue

Please check:
- the [README](README.md)
- existing issues and pull requests
- relevant docs in the `docs/` folder

## Best kinds of support requests

Helpful support requests usually include:
- what you were trying to do
- what happened instead
- exact steps to reproduce
- screenshots or logs
- device, OS, or runtime details

## Project focus

SmartTime AI is an education-focused open-source project. Support priorities may favor bugs and improvements that directly affect timetable generation, school workflows, and contributor onboarding.
